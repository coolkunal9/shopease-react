# 🛍️ ShopEase — React E-Commerce Store

A modern, fully functional e-commerce frontend built with React + Vite.

## ✨ Features

- 🛒 **Shopping Cart** — Add, remove, update quantity with localStorage persistence
- 🔍 **Search & Filter** — Real-time search + category filter (Tech / Fashion)
- 🔔 **Toast Notifications** — Animated feedback on add to cart
- 📱 **Fully Responsive** — Mobile-first design
- ✅ **Contact Form** — Client-side validation with error messages
- 💾 **Cart Persistence** — Cart saved to localStorage on refresh

## 🚀 Tech Stack

- React 18 (Hooks — useState, useEffect, useRef)
- Vite (build tool)
- CSS-in-JS (inline styles + style tag)
- localStorage API
- Google Fonts (Syne + DM Sans)

## 🏃 Run Locally

```bash
git clone https://github.com/coolkunal9/shopease-react
cd shopease-react
npm install
npm run dev
```
